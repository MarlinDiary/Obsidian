## 修饰符

| 修饰符                                 | 作用           |
| ----------------------------------- | ------------ |
| .preferredColorScheme(.dark)<br>    | 视图始终处于暗模式    |
| .scrollBounceBehavior(.basedOnSize) | 在没有滚动内容时禁用弹跳 |
| .localizedStandardContains("")      | 搜索字符串，但忽略大小写 |
|                                     |              |
